Description back to ToC

This vocabulary represents online API model from OSDM.

The scope of this vocabulary is defined with the following information:

  • Offer Parts
  • Trip Offers
  • Fare product
  • Offer
  • Trip
  • Stop
  • Location
  • Segment
  • Trip
  • Passenger

Some diagrams that are part of the model have been added.

Examples

This example represents a trip offer from Chamartin to Aranjuez Renfe C-3.
